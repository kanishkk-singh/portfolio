# Assets Folder — Yahan apni files daalo

## 📸 Avatar Photo (Left panel mein dikhega)
- File name: `avatar.jpg` ya `avatar.png`
- Size: 200x200px recommended
- Square crop karo photo ko
- Example: `assets/avatar.jpg`

## 🖼️ Project Screenshots (Project ke saath thumbnail)
- `skillswap.png`    — SkillSwap ka screenshot
- `deadlinedock.png` — DeadlineDock ka screenshot
- `sidebyside.png`   — SideBySide ka screenshot
- Size: 400x280px recommended (16:11 ratio)
- Browser mein apna project kholo → F12 → Screenshot tool use karo

## 📄 CV/Resume PDF
- File name: `Kanishk_Kumar_Singh.pdf`
- Jab bhi CV update karo, sirf yeh file replace karo
- DOWNLOAD_CV.PDF button automatically kaam karega

## Quick Steps:
1. Apni photo → assets/avatar.jpg
2. Project screenshots → assets/skillswap.png etc
3. CV PDF → assets/Kanishk_Kumar_Singh.pdf
4. Done! Portfolio mein sab automatically show hoga.
