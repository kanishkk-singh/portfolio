## How to take screenshots:
1. Open your live project in Chrome
2. Press F12 → three-dot menu → "Capture screenshot"
3. Or: Ctrl+Shift+P → type "screenshot" → "Capture full size screenshot"
4. Save with the exact filename above in this folder

## Recommended size: 1280x800px or 1440x900px
