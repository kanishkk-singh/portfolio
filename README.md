# Kanishk Kumar Singh — Portfolio

Terminal-style developer portfolio inspired by Prithiviraj Elango.

## 🚀 Live Deploy Options

### Option 1: Vercel (Recommended — Free)
1. Push this folder to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import your GitHub repo
4. Click Deploy — done!

### Option 2: Netlify (Free)
1. Go to [netlify.com](https://netlify.com) → Add new site
2. Drag & drop this entire folder
3. Live in seconds!

### Option 3: GitHub Pages (Free)
1. Push to GitHub repo (name it `username.github.io`)
2. Settings → Pages → Source: main branch
3. Live at `https://kanishkk-singh.github.io`
